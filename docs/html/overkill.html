overkill.overkill module¶

Main module. All user interactions to distribute tasks should use this module. See the ClusterCompute for more details on distributing tasks.

class overkill.overkill.ClusterCompute(n_workers: int, master_address: Tuple[str, int])[source]¶

Bases: object

The ClusterCompute class acts as the main interface between the Master server and the user. use this class to distribute an array over a cluster of computers given a function.

Parameters

  • n_workers (int) – number of workers to use to distribute (array is distributed evenly accross workers)

  • master_address (Tuple[str, int]) – A tuple of (ip, port) e.g. (“localhost”, 5555). Use the .get_address() class member of the Master class to get the address
